
# 做多了就是套路

## 数据描述：
	-分类还是连续变量
	-数据的离散和集中程度 -barplot() plot(density())
	-缺失值占比
	-考虑缺失值填补


## 缺失值：
	-直接删除进入模型
	-填补为0进入，模型无法判别给出的答案
	-利用插值的方法进入，模型给出的答案
	-标签填补
	

## 不平衡数据的处理：
	-欠采样
	-过采样
	-阀值移动
	-填补，打标签


## 特征选择：
	- 数据要进行无量纲化，标准化，区间缩放法，标准与归一化
	- 定量特征二值化
	- 方差选择
	- 相关系数法
	- 卡方检验
	- 互信息法
	- 递归特征消除法
	- GBDT  50-60到几百个
	- RF随机森林 几百个特征
	- SVM
	- 分为特征子集的产生过程，评价函数，停止准则，验证过程
		特征子集产生：穷举，启发式，随机的提取
		改变特征空间情况：去除相关性比如：PCA, 傅立叶变换，小波变换
		评价的时候：过滤器模型，封装模型，混合模型
	- 进行搜索，典型的当长度不在考虑的时候，向前（SFS , Sequential Forward Selection）和
	 向后（Sequential Backward Selection）的进行剔除，同时加入时间内外
	 或样本内外的选择来表述模型的稳定程度
	- 评价函数：独立准则，关联准则（开销大）模式分类的时候比较好
	  一般有：1.卡方检验（过滤），2.相关性（过滤），3.距离（过滤），4.信息增益（过滤），5.分类器错误率（封装器）


## 模型：
	- 监督，无监督，分类，回归，降维，聚类，数值分析，路径规划
	- 模型的表达式算法（重要），毕竟每一次运算只是进行优化，实际能不能逼近目标函数的解理论上证明的被封装了。
	- LM,LAR,LASSO, LR,KNN,DTR,RF,SVM,SVC,POLYLM,PCA,IDA,GBDT,ADABOOST,EM,NB,Ridge,ElasticNet, LVQ,SOM
	- CART,C50,CHAID,MARS,A/B,BAYENET,NNET

## 检验：
	- 分类器特征，F,准确性，精准度，ROC,TP,FP,FN,TN,AUC
	- 拟合：R^2,系数显著性，残差图，卡方值，SSE
	- K-FOLD检验，测试训练，leave-one，时间内外，随机抽样,衰减程度
	- 模拟数据进入得出的是学习曲线，随着数据的增加模型的准确性在逐渐增强

## 过拟合：
	- 每一次重新SHUFFLE进行训练
	- 加入正则项还有惩罚项
	- early stop
	- drop out
	- 噪音清理	
	- 提高或减少有效的标签

## 欠拟合：
	- 添加其他特征
	- 添加变化的特征
	- 减少正则化参数
	- 集成方法
